## 🚀 **All-in-one platform for property acquisition tracking**

DealPilot is the premium mobile-first application that pilots every step of your property purchase. From first offer to final signature, we handle the details so you can focus on what matters: owning with confidence.

### **Core Features**

**🏠 Multi-Project Management**
- Track multiple acquisitions simultaneously (Villa Marrakech, NYC Penthouse, London House...)
- 12 property types with intelligent filtering by region
- Smart country/city/currency selection workflow

**💰 Real-Time Currency Conversion**  
- Live exchange rates from European Central Bank
- 30+ global currencies supported
- Local property currency + display currency for totals

**📋 Timeline Management**
- Pre-configured acquisition milestones
- Visual progress tracking with completion status
- Calendar integration ready

**💸 Budget & Cost Tracking**
- Multi-currency expense categories
- Real-time conversion and totals
- Professional PDF/Excel exports

**📁 Document Organization**
- Categorized document management (Pending/Signed)
- Upload and tracking system
- Quick access to critical files

### **Mobile-First Design**

- **Dark premium interface** with gold accents
- **Bottom navigation** for mobile optimization
- **Card-based layout** with smooth animations
- **Typography**: Poppins for modern readability

## 🛠 **Technology Stack**

- **Frontend**: HTML5, CSS3 (Mobile-First), JavaScript ES6+
- **APIs**: Frankfurter (ECB) + fallback Open Exchange Rates  
- **Storage**: Modern Storage API + localStorage fallback
- **Deployment**: Vercel + GitHub
- **Performance**: Progressive Web App ready

## 🚀 **Quick Start**

### **Deploy to Vercel**
```bash
# Clone repository
git clone https://github.com/[username]/dealpilot.git
cd dealpilot

# Connect to Vercel
vercel --prod
```

### **Local Development**
```bash
# Simple HTTP server
npx http-server . -p 3000

# Or use any static server
python -m http.server 3000
```
